# Solutions

В этой папке размещаются примеры решений лабораторных работ.

## Доступность

Решения становятся доступными **после дедлайна** соответствующего занятия.

## Структура

```
solutions/
├── 01_cnn_introduction_solution.ipynb    # После занятия 1
├── 02_overfitting_solution.ipynb         # После занятия 2
├── 03_transfer_learning_solution.ipynb   # После занятия 3
├── 04_distillation_solution.ipynb        # После занятия 4
├── 05_yolo_solution.ipynb                # После занятия 5
├── 06_vit_solution.ipynb                 # После занятия 6
├── 07_autoencoder_solution.ipynb         # После занятия 7
├── 08_vae_solution.ipynb                 # После занятия 8
├── 09_gan_solution.ipynb                 # После занятия 9
└── 10_diffusion_solution.ipynb           # После занятия 10
```

## Предупреждение

Использование готовых решений без самостоятельной работы не способствует усвоению материала. Рекомендуется сначала выполнить задание самостоятельно, а затем сравнить с примером решения.
